Quick Installation Commands

One-Line Install Commands

These commands allow you to quickly set up your server with a single command. Simply copy and paste them into your terminal.

Basic Server Setup (s1.sh)

Sets up essential packages, configures firewall, secures SSH, and sets up automatic updates.

curl -sSL ls.r-u.live/sh/s1.sh | sudo bash
Mail Server Setup

Installs and configures a complete mail server with Postfix, Dovecot, and webmail.

curl -sSL ls.r-u.live/sh/setup-mail-quick.sh | sudo bash
DNS Server Setup

Sets up BIND DNS server in master, slave, or caching mode.

curl -sSL ls.r-u.live/sh/setup-dns-quick.sh | sudo bash

PowerShell Install (Windows)

Windows Server Setup

For Windows systems, use PowerShell to download and execute scripts for basic server configuration.

iwr -useb ls.r-u.live/sh/windows-setup.ps1 | iex
Windows Mail Server Setup

Sets up a mail server on Windows using hMailServer with optional webmail.

iwr -useb ls.r-u.live/sh/windows-mail-setup.ps1 | iex
Windows DNS Server Setup

Installs and configures Windows DNS Server role with zone configuration.

iwr -useb ls.r-u.live/sh/windows-dns-setup.ps1 | iex

Security Note

Always review scripts before running them with elevated privileges. You can inspect any script by opening its URL in your browser.

Example: https://ls.r-u.live/sh/s1.sh
