Server Setup Scripts

Automated scripts for server deployment and configuration

Overview

A collection of scripts for automating server setup and configuration across different operating systems, focusing on Ubuntu/Debian systems.

These scripts provide comprehensive solutions for:

  • Base server setup with essential packages and security
  • Complete mail server configuration with multiple component options
  • BIND DNS server setup for domain management

Server Setup

Usage

chmod +x setup-server/setup.sh
./setup-server/setup.sh

The server setup script automates the installation of:

  • Git and GitHub CLI
  • Node.js (LTS version)
  • PostgreSQL and MySQL databases
  • Docker with Docker Compose
  • Security tools like Fail2Ban
  • SSL tools (OpenSSL and Certbot)

Mail Server Setup

Usage

# Basic setup
./setup-mail/setup-mail.sh --domain example.com

# Advanced setup with options
./setup-mail/setup-mail.sh \
  --domain example.com \
  --hostname mail.example.com \
  --imap-server courier \
  --webmail-type roundcube

Configure your mail server with multiple component options:

  • SMTP Server: Postfix (default)
  • IMAP/POP3: Choose between Dovecot or Courier
  • Webmail: Roundcube or SquirrelMail
  • Security: SpamAssassin, ClamAV, DKIM, SPF

DNS Server Setup

Usage

# Setup as Primary DNS server
./setup-dns/setup-dns.sh --domain example.com --server-ip 192.168.1.10

# Setup as Secondary DNS server
./setup-dns/setup-dns.sh --setup-slave --domain example.com --master-dns 192.168.1.10

# Setup as Caching DNS server
./setup-dns/setup-dns.sh --setup-caching

Configure BIND DNS server in various roles:

  • Master/Primary DNS server - Authoritative for your domain
  • Slave/Secondary DNS server - Backup with automatic zone transfers
  • Caching-only DNS server - For improved lookup performance

Getting Started

To use these scripts:

  1. Clone the repository: git clone https://github.com/yourusername/server-setup-scripts.git
  2. Make scripts executable: chmod +x setup-*/setup*.sh
  3. Run the appropriate script with your desired options

Each script includes detailed help. Just run with --help to see available options.
