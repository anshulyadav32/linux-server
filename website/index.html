Installation Methods

⚡ Quick Remote Install

One-line installation from live repository:

curl -sSL https://raw.githubusercontent.com/anshulyadav32/linux-setup/main/s1.sh | sudo bash
  • ✅ Fastest installation method
  • ✅ Always uses latest version
  • ✅ Automatic dependency handling
  • ✅ No local download required

📦 Local Installation

Clone and install locally for full control:

git clone https://github.com/anshulyadav32/linux-setup.git
cd linux-setup
sudo ./install.sh
  • ✅ Full control over installation
  • ✅ Installs all 25+ components
  • ✅ Customizable configuration
  • ✅ Production ready

Maintenance & Update

📦 Update System

# Update Linux Setup
cd /path/to/linux-setup
git pull origin main
sudo ./install.sh --update

# Or use quick update
curl -sSL https://raw.githubusercontent.com/anshulyadav32/linux-setup/main/s1.sh | sudo bash

🔧 Fix Common Issues

Permission Issues:
chmod +x *.sh modules/**/*.sh
Service Problems:
sudo systemctl restart apache2
SSL Issues:
sudo certbot renew --dry-run
