Installation Methods

⚡ Quick Remote Install

One-line installation from live repository:

Bash
curl -sSL https://raw.githubusercontent.com/anshulyadav32/linux-server/main/install.sh | sudo bash
  • ✅ Fastest installation method
  • ✅ Always uses latest version
  • ✅ Automatic dependency handling
  • ✅ No local download required

📦 Local Installation

Clone and install locally for full control:

Bash
git clone https://github.com/anshulyadav32/linux-server.git cd linux-server sudo ./install.sh
  • ✅ Full control over installation
  • ✅ Installs all 25+ components
  • ✅ Customizable configuration
  • ✅ Production ready

🔧 Individual Module Installation

To install a specific module, run its install script directly from the modules folder:

Webserver:
sudo bash modules/webserver/install.sh
Database:
sudo bash modules/database/install.sh
DNS:
sudo bash modules/dns/install.sh
Firewall:
sudo bash modules/firewall/install.sh
SSL:
sudo bash modules/ssl/install.sh
Backup:
sudo bash modules/backup/install.sh

Modules Quick Guide

Module Quick Commands Purpose

Maintenance & Update

📦 Update System

Update Commands
# Update server components sudo ./update-server.sh # Update with backup sudo ./update-server.sh --backup # Preview updates (dry run) sudo ./update-server.sh --dry-run

🔧 Fix Common Issues

Permission Issues:
chmod +x *.sh modules/**/*.sh
Service Problems:
sudo systemctl restart apache2
SSL Issues:
sudo certbot renew --dry-run
